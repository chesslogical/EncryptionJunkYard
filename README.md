Search git for rust encryption apps and you will find a bloated pile of apps, all with numerous files. I thought that is just plain silly. Here, there are lots of apps , the full code for each app is in its main.rs . As such, it is incredibly easy to feed the main.rs of any app to ai (like chatgpt or grok or whatever) in order to audit, change anything, fix, update or whatever. You could not ask for more from a junkyard of superior code.
