At the time i made this, zig sux for password based encryption. So here is a c app, it can be compiled in zig with the command  zig cc encrypt.c -o encrypt.exe  

usage- just encrypt.exe [any.file]  it automatically encrypts or decrypts the file for you. 

Zig sort of has the chicken and egg problem. It is not developed much yet, breaking changes every new versioon. Without tons of users testing everything, the dev is super slow.
